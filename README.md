
















# bloggify-shortcode

Shortcode plugin for Bloggify.




## Installation

```sh
$ npm i bloggify-shortcode
```









## Example






```js
"use strict";

{
    // Replace ":smile:" everywehre
    ":smile:": ":D"
}

```
